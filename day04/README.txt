用法: python main.py
注意：所有的文件需要在同一级目录下。。。

===========================
文件介绍
===========================
main.py  潘咳松游戏程序逻辑调用主文件。
GameMainClass.py  游戏程序功能类的主文件，这里存放了三个类，创建人物的类，创建账号的类，场景类。
userlist.pkl 程序用户初始化文件，这里还是阿尔法版本，由于时间问题，没有写删除用户的程序，不要动里面的数据，不然主程序会有问题。
lock.txt 用户锁文件，此次程序没有写解锁功能，用户如果锁定，想解锁可直接在lock.txt文件中把内容删除即可。
useinfo 目录 ，这个目录下存放了每个账号的信息，每个账号目录下，还有角色的信息。

===========================
功能介绍
===========================
1.支持创建账号
2.进入账号后可创建多个角色
注意：新角色进入游戏后默认为等级1级，金钱为10000。默认工作1个月薪资为1000，等级为1，则为1000，等级为2，则为2000，等级越高，薪资越高。
3.角色进入游戏后，有以下几项娱乐:
①学习
学习会扣除金钱，但是等级会提升
②找性福
这个需要一定的等级，才能娱乐。
③工作
④交友
也是花钱买等级，跟学习差不多性质。
⑤想不想一夜暴富
这个就是赌博，创业的成功率为10%，一旦成功，功成名就都有了。股票则为0.1-2之间的资金膨胀或缩水，如果赔了还要降级，谁叫你不误正业的。
4.角色支持读档。
